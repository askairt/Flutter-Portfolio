# Flutter Responsive Web Portfolio

Star ⭐ the repo if you like what you see😉.


#### How to Install?
```
git clone https://github.com/askairt/flutter-portfolio.git
cd flutter-portfolio
flutter packages get
flutter run -d chrome
```
